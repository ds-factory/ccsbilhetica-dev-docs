# Pré‑requisitos

| Ferramenta | Versão mínima |
|------------|---------------|
| Node.js    | 20.x          |
| Yarn (Corepack) | 1.22 |
| PHP        | 8.2 |
| Composer   | 2.x |
| Docker & Docker Compose | latest |
| PostgreSQL | 15 |
| Git        | 2.40+ |

> **Windows**: ative o _ExecutionPolicy_ do PowerShell para permitir scripts:  
> `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned`
